# 服务器内存类型与代际

本文内容迁移自原博客《IT基础知识百科》中内存相关部分，整理服务器常见内存类型与代际。

---

## 内存类型

| 类型 | 说明 |
|------|------|
| non-ECC | 普通家用内存 |
| ECC UDIMM | 可用于服务器 |
| RDIMM | 大容量服务器 |
| LRDIMM | 超大容量服务器 |

---

## 内存代际

| 代 | 频率 |
|----|--------|
| DDR3 | 800–2133 |
| DDR4 | 1600–3200 |
| DDR5 | 6000–8000MT/s |

---

## 备注

- Intel 非至强平台通常不支持 ECC  
- AMD Ryzen 部分支持 ECC
