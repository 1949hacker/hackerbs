# hackerbs Knowledge Base

A long-term evolving engineering knowledge system focused on Data Center, IT Infrastructure, and Platform Engineering.

这是一个面向 **数据中心 / IT 基础设施 / 系统工程 / 云与平台工程** 的长期工程知识体系，而不仅仅是一个博客。

## 🎯 项目定位

本项目旨在构建一个：

- 📚 **可积累的工程知识系统**
- 🧠 **可复用的工程方法论与实践集合**
- 🏗️ **面向真实工程场景的技术文档体系**
- 🔁 **可持续演进的技术知识库**

关注领域包括但不限于：

- 数据中心与 IT 基础设施
- 网络与通信系统
- 服务器与硬件体系
- 存储与系统可靠性
- 云平台与平台工程（Platform Engineering）
- 运维自动化与 SRE 思维

---

## 🧱 技术架构

本知识库基于以下技术构建：

- 📄 Static Site Generator: **Quartz**
- ✍️ Markdown 驱动内容体系
- 🌐 部署：Cloudflare Pages
- 🧩 支持 Obsidian 风格双链 & 图谱
- 🎨 自定义主题 + Google Fonts
- 🌗 支持 Light / Dark 模式
- 📡 自动生成 RSS / Sitemap
